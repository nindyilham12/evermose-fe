## Getting started

- Install dependencies

  ```bash
  npm install
  ```

- Start `Next` development server

  ```bash
  npm run dev
  ```

  This will start a web server at `http://localhost:3000`

## Build for production

- Build the `Next` app

  ```bash
  npm run build
  ```

  This will generate directory `.next`
